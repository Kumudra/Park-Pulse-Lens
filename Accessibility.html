Accessibility Statement

At Park Pulse Lens, we believe everyone should have equal access to nature and community. This statement outlines our commitment to digital and physical accessibility.

Last Updated: January 15, 2024 | WCAG 2.1 AA Compliant

Our Accessibility Commitment

Creating inclusive digital experiences and accessible park environments for all

Inclusive Design

We design our digital platform and park events with diverse abilities in mind, following universal design principles that benefit everyone.

Continuous Improvement

Accessibility is an ongoing journey. We regularly review and enhance our platform based on user feedback and evolving standards.

User Partnership

We collaborate with accessibility experts and community members with disabilities to ensure our solutions meet real needs.

Digital Accessibility Features

Our website is designed to be accessible to people with diverse abilities

Website Accessibility

  • Keyboard Navigation

    All interactive elements can be accessed using keyboard-only navigation. Use Tab to navigate and Enter/Space to activate.

  • Screen Reader Compatibility

    Proper ARIA labels, semantic HTML, and descriptive alt text for all images. Compatible with JAWS, NVDA, and VoiceOver.

  • Text Resizing

    Text can be resized up to 200% without loss of content or functionality. Use browser zoom or our accessibility widget.

  • Color Contrast

    Minimum 4.5:1 contrast ratio for normal text and 3:1 for large text, exceeding WCAG AA requirements.

  • Focus Indicators

    Clear visual focus indicators for keyboard users with high contrast outlines on interactive elements.

Mobile Accessibility

Our website is fully responsive and accessible on mobile devices:

  • Touch targets minimum 44x44 pixels
  • Responsive design for all screen sizes
  • Mobile screen reader compatibility
  • Gesture alternatives available
  • Orientation flexibility

WCAG 2.1 Compliance

Level AA
Full Compliance

We meet all WCAG 2.1 Level AA success criteria for web accessibility.

Perceivable

Content is presented in ways users can perceive. Text alternatives, captions, and adaptable presentation.

Operable

Interface components are operable through various methods. Keyboard accessible, enough time, no seizures.

Understandable

Content and operation are understandable. Readable, predictable, input assistance.

Robust

Content is robust enough for current and future technologies. Compatible with assistive technologies.

Park Event Accessibility

Ensuring our community events are accessible to everyone

Accessible Park Initiatives

We're committed to making park revitalization events inclusive for participants of all abilities:

Physical Accessibility Features:
  • Accessible parking and drop-off zones at event locations
  • Paved or firm pathways to event areas
  • ADA-compliant portable restrooms
  • Seating areas with back support
  • Shaded rest areas with accessible tables

Inclusive Activities

We offer a range of activities suitable for different abilities:

  • Table-top planting and gardening stations
  • Seated sorting and preparation tasks
  • Educational workshops with sign language interpretation
  • Guided nature tours with audio description
  • Adaptive tools and equipment
Sensory Considerations

Quiet zones, noise-reducing headphones available, visual schedules, and sensory-friendly event options.

Communication Access

ASL interpreters, real-time captioning, communication boards, and visual instruction materials.

Personal Support

Trained accessibility volunteers, personal care assistant accommodations, and flexible participation options.

Known Limitations

While we strive for full accessibility, we acknowledge some current limitations:

Third-Party Content

Some embedded maps or external content may not be fully accessible. We provide alternative text descriptions.

Legacy Documents

Older PDF documents may not be fully tagged. We're working to update these and provide accessible alternatives.

Park Terrain

Some natural park areas have terrain limitations. We always provide accessible alternative locations.

Technical Information

Browser Compatibility

Our website is compatible with the latest versions of:

  • Chrome with NVDA/JAWS
  • Firefox with NVDA
  • Safari with VoiceOver
  • Edge with Narrator
Assistive Technologies Tested
  • Screen Readers: JAWS, NVDA, VoiceOver
  • Screen Magnifiers: ZoomText, Magnifier
  • Speech Recognition: Dragon NaturallySpeaking
  • Alternative Input: Switch devices, eye tracking
Accessibility Testing

We conduct regular accessibility testing using:

  • Automated testing tools (axe, WAVE)
  • Manual keyboard testing
  • Screen reader testing
  • User testing with people with disabilities
  • Color contrast analysis

  • Large print documents (18pt+)
  • Braille materials (advance notice required)
  • Audio descriptions of visual content
  • Plain language summaries
  • Video with captions and audio description

Request formats at least 7 days in advance.
